# Supply-Chain-Inventory-Optimization-
supply chain inventory optimization analysis using Microsoft Excel, Python(Pandas), SQL, and Power BI

Problem

Inefficient inventory management can lead to stockouts, overstocking, and increased operational costs. This project analyzes supply chain data to optimize inventory levels.

Objective

Identify demand patterns, supplier performance, and inventory trends to improve stock management.

Tools Used

- Excel
- SQL
- Python (Pandas, Matplotlib)
- Power BI

Analysis Steps

1. Data cleaning and preprocessing using Python
2. SQL queries to analyze stock levels and supplier performance
3. Visualization and dashboard creation using Power BI

Key Insights

- Certain products experience frequent stockouts
- Some suppliers have longer delivery times
- Demand shows seasonal trends affecting inventory planning
